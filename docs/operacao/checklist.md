# Checklist

Antes de treinar de verdade:

- Ambiente virtual ativado.
- Dependencias instaladas.
- Modelo escolhido conforme VRAM disponivel.
- Dataset JSONL validado.
- Exemplos sem dados sensiveis desnecessarios.
- Config apontando para o dataset correto.
- `output_dir` separado para cada experimento.
- Primeiro treino curto para validar o pipeline.
- Teste de inferencia feito antes de aumentar escala.
